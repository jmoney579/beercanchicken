Portfolio 3.0

This website was created roughly a year ago. The theme for this site is to break the molds. Typical single page sites scroll vertically with multiple "stops" to indicate changes in topics. This is also a single page site; however, it scrolls horizontally, until reaching it's mobile state where it returns to the typical scroller. Utilizing multiple jquery plugins, I was able to create a site which broke molds after it's launch.
