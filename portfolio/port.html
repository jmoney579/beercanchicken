Online Portfolios

Portfolio 4.0

Portfolio version 4.0 was developed to take advantage of simplistic design princliples. Using this approach, I created a site which focused more on the presentation of content than on wowing the user with design asthetics. Although this was the case, I still maintained a level of creative design to maintain user engagment. Version 4.5, which is currently in use, is a refresh of this design utilizing a "clean lines" approach and vertical navigation to create a one of a kind experience that not only meet currect design priciples but also surpasssed imerging trends.

Portfolio 3.0

This website was created roughly a year ago and was designed to break molds. Typical single page sites scroll vertically with multiple "stops" to indicate changes in topics. This is also a single page site; however, it scrolls horizontally, until reaching it's mobile state where it returns to the typical vertical scroller.

Portfolio 2.0

I created this website approximately 2 years ago. Since the typical half-life for a website is anywhere between a year and three, I felt that it was due for retirement. My intentions with this site were to be bold and carry an "in your face" style. It worked well, but the world has moved towards more minimalistic designs.

You also noticed that version 2.0 is showcased, but a 1.0 is not. Version 1.0, though complete in design, was never complete in content or saw the light of day. However, if you're really curious you might find it lurking around here somewhere.
