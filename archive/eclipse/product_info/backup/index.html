Three Convenient Screens to View/Enter Data


Full Screen

This screen avoids having too much empty space or data overload by recording just the essentials.





Quick Screen

This screen allows for the enter and browse essential information, especially helpful when entering large amounts of data in a short time frame.





Browse Screen

This screen allows for effortless searching and browsing. Searching using one or more fields, ensures that you find the correct information every time. Once you find the record that you're looking for, simply click the Full or Quick Screen button to view its details.
