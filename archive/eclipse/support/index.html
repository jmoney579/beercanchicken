Support for EGEMS


ECLIPSE has done it's best to make EGEMS an intuitive and user-friendly system. If you have a question, read through these tips to see if EGEMS or FileMaker can help you answer it!


  • Hover your mouse over the object (field, label, button, etc) you are having trouble with. EGEMS has many help dialogues, and one might just pop up that answers your question.

  • See if any instructional text was overlooked. Instructions are usually found near the bottom of the page, and may require scrolling to see fully.



If you still have a question, see if it pertains to one of the topics below. FileMaker provides free support for any of these topics. The toll-free number to contact FileMaker customer service is: 1-800-325-2747, or go to FileMaker Support.


  • Installation/Uninstallation

  • LaunchActivation

  • Updaters 

  • FileMaker Network Sharing

  • Instant Web Publishing

  • Opening FileMaker Authenticated Database Files

  • xDBC

  • External SQL Data Sources

  • Solution Conversion



If the problem you are having is still not resolved ECLIPSE is happy to help! Choose from the options below to receive fast and friendly customer support.*


  • Use the contact form to send us an inquiry

  • Email us at customersupport@egemsinfo.com

  • Give us a call at 1-540-328-0484
