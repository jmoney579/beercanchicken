Three Convenient Screens to View/Enter Data

Full Screen

This screen allows you to enter and browse all a material's information. Each category of information has its own tab so you can easily access or enter information without feeling overwhelmed.





Quick Screen

With this screen you can view or input just the essential information for a material. This is especially helpful when entering in large amounts of data in a short time frame.





Browse Screen

This screen allows for effortless searching and browsing. Searching using one or more fields, ensures that you find the correct information every time. Once you find the record that you're looking for, simply click the Full or Quick Screen button to view its details.

Simultaneous Multi-Field Search

Searching multiple fields at one allows you to quickly obtain the information you are looking for. The first screen in this example searched one field returning 134/1152 results. When multiple fields are searched (screen two), the results are narrowed to 3/1152 (screen three).








Combination Data Import and Drop-down Menu Updates

With EGEMS there is no need to re-type data if it is already digitized. Simply click on the import button in the navigation bar, and follow the instructions to easily import your data! If importing from excel, the column names will conveniently appear when you are setting up the import process so you can match them to fields in EGEMS.

After importing your data EGEMS will automatically go through each of the drop-down menus and update it with any new values. EGEMS will also delete any duplicate values so during data entry you can quickly find the value you are looking for!

Revolutionary Research Product Data Preservation

Research products are geological materials
that can be used for research or were derived from research. Examples of research products
include: thin sections (polished, or covered), billets, experimental powders, synthesized materials, etc.
When information for these are fully filled in, they act as a recipe that can be made again in the future. Most of the fields shown are new to geological databases everywhere.







User Accessible Revision History

An exclusive user function of EGEMS is the Change History. This Screen records who changed which record and when. If any main record field has been altered in any way a pop-up message is triggered, that says “Do you want to save these changes?” (screen one). You then have the option of selecting “no” and reverting the record to the way it was when it was originally opened, or “yes”, which will record the change in the history. In screen two the yellow brackets are highlighting changes made to this record.







Easily Import or Export Images & Documents

Insert Image

"A Picture is Worth One-thousand Words"
EGEMS makes it very easy to insert images and can communicate much information about a material. The photos on these screens include a petrographic image taken from down a microscope, a backscatter electron (BSE) image, and two element maps. 



Image Browse

An image browse is available for each material type. This can be especially useful when searching for a quality specimen, or completing an inventory. Since every field in EGEMS is searchable, the summary field in the image browse can be easily queried to find any record.



Insert Documents


This is a flow chart for inserting a document. EGEMS allows many files of different types to be attached to one record. In this figure alone there three file types: an Excel file, Word doc, and GIF. This can be especially useful for Research Products with analytical data. 







Innovative Citation Records

This screen is demonstrating the ability to record citations, another unique feature of EGEMS. Citations are especially important for materials that have been referenced in publications, however this function can be used to record articles that are related to the material in other ways as well.
